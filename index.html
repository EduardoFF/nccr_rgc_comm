Introduction

The purpose of this document is to describe the framework that is used to build and maintain a multi-hop wireless network infrastructure for the NCCR Grand Challenge on Rescue Robotics.

Proposed hardware

  • ODROID U3 or C1(+) with Ubuntu
  • XBee-PRO 868
  • TP-LINK TL-WN722N
  • Camera, GPS Dongle (optional)

Installing dependencies

LCM:

  • https://github.com/lcm-proj/lcm/releases/download/v1.3.1/lcm-1.3.1.zip

Click:

  • git clone https://github.com/EduardoFF/click.git
  • cd click
  • ./configure
  • make -C ./userlevel MINDRIVER=RNP_BRIDGE_PKG

libxbee

  • git clone https://github.com/attie/libxbee3.git
  • follow the indicated installation procedure

XBee bridge:

  • git clone https://github.com/EduardoFF/rnp_xbee_bridge.git
  • cd rnp_xbee_bridge
  • mkdir build
  • cd build
  • cmake ../src
  • make end_node
